# 🕷️ Web Crawler & Web Structure Miner

A simple and interactive web crawler built with **BeautifulSoup**, **Streamlit**, and **NetworkX**, designed to extract links from a website and visualize the structure of internal web pages.

---

## 🔍 Features

- Crawl internal links from a seed URL
- Limit crawl depth (max pages to visit)
- Automatically extract and display page-to-page links
- Visualize the website's link structure using a graph
- Interactive user interface built with Streamlit
